# FullScreen

스크롤없는 SPA앱의 wrapper, App바로 안쪽에 한번만 선언하며 그 높이는 `100vh`(클라이언트 화면 높이의 100%) 너비는 `100vw`(클라이언트 화면 너비의 100%)로 고정됩니다.
Flexbox Container를 상속받습니다. 

## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">Flex</Chip>
</Flex>
</App>

## Properties

|Property|Type|Default|
|-|-|-|
| flexDirection | row\|column | row |
| justifyContent | flex-start\|flex-end\|center\|space-between | flex-start |
| alignItems | flex-start\|flex-end\|center\|stretch | flex-start |
| flexWrap | wrap\|no-wrap | no-wrap |
