# Hide

Responsive하게 display속성을 켜고 끄는 wrapper. 속성을 여러개 동시에 넣어줄 수 도 있습니다. 


## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">None</Chip>
</Flex>
</App>



## Properties

|Property|Type|Default|
|-|-|-|
|mobile| boolean | false |
|tablet| boolean | false |
|desktop| boolean | false |
