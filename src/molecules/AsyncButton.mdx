# AsyncButton

버튼입니다. onClick Prop에 콜백함수를 넣어주세요.

## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">Box</Chip>
</Flex>
</App>

## Properties

|Property|Type|Default|
|-|-|-|
| color | string | white.standard |
| height | ResponsiveValues | [52px, 60px] |
| width | ResponsiveValue | fit-content |
| fontSize | ResponsiveValues | [h5, h3] |
| hoverColor | String | color.blue.deep |
| focusColor | String | color.blue.dark |
| bg | String | blue.royal |
| onClick | function | ()=>{} |
| disabled | boolean | false |
| loading | boolean | false |

- hoverColor, focusColor는 theme적용이 안되므로 theme을 import해서 color.*으로 사용했습니다.

<Playground>
