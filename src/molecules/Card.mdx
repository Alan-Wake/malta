# Card

카드 컨테이너 입니다. Column Flexbox를 사용하며 기본적으로 boxShadow와 hover시의 boxShadow가 적용되어있습니다. 오버라이딩할 수 있습니다.

## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">Flex</Chip>
</Flex>
</App>

## Properties

|Property|Type|Default|
|-|-|-|
|width|ResponsiveValue|[100%, 300px]|
|height|ResponsiveValue|auto|
|borderRadius|ResponsiveValue|3px|
|shadow|ResponsiveValue|2|
|hoverShadow|ResponsiveValue|4|
