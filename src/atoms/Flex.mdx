# Flex

<Image src="https://cdn-images-1.medium.com/max/2000/1*wyQwuPQq3CBP4p8lDX4sGQ.gif"/>

Flex 레이아웃을 그리기 위한 부모 컨테이너입니다. CSS구이조상 부모이자 자식으로 사용하는 것도 가능하다. 
다만 하위 컴포넌트가 하나일 경우  `<Box>`를 사용하는 것이 옳습니다.

- [Flexbox — The Animated Tutorial - medium](https://medium.com/@js_tut/flexbox-the-animated-tutorial-8075cbe4c1b2)
- [A Complete Guide to Flexbox - CSS tricks](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)
- [CSS flexbox cheatsheet](https://devhints.io/css-flexbox)

## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">Box</Chip>
</Flex>
</App>

## Properties

|Property|Type|Default|
|-|-|-|
| flexDirection | row\|column | row |
| justifyContent | flex-start\|flex-end\|center\|space-between\|...| flex-start |
| alignItems | flex-start\|flex-end\|center\|stretch\|... | flex-start |
| order | number | 0 |
| flexWrap | wrap\|no-wrap | no-wrap |
