# Box

가장 기본이 되는 Box 엘레먼트. 컨테이너로 사용하면 안됩니다. 즉 하나 이상의 자식노드를 가질 수 없습니다.
컴포넌트의 wrapper로 사용하여 width, height, margin, padding, background, color등의 속성을 적용시키는 용도로 사용합니다.
특히 Box안에 text를 집어 넣을 경우에는 `<Text>`를 하위 컴포넌트로 두는게 좋습니다.

## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">None</Chip>
</Flex>
</App>

## Properties

|Property|Type|Default|
|-|-|-|
|space|ResponsiveValue|0|
|width|ResponsiveValue|100%|
|height|ResponsiveValue|auto|
|color|ResponsiveValue|inherit|
|justifySelf|ResponsiveValue|none|
|alignSelf|ResponsiveValue|none|
|order|number[]|none|
|border|ResponsiveValue|none|

## Best Practices

<App>
  <Chip bg="#00AA00"color="#FFF" width="100px">Do</Chip>
</App>

```javascrip
//하위 컴포넌트의 margin, padding, flex-child-props, color, bg를 설정하는 용도로 사용 
<Box><Image...></Box>
<Box><Text...></Box>
```

<App>
  <Chip bg="#AA0000"color="#FFF" width="100px">Don't</Chip>
</App>

```javascript
//하위 컴포넌트 여러개를 묶는 부모 컴포넌트로 사용하면 안됩니다. 대신 <Flex>를 사용하도록 합니다. 
<Box>
  <Box>...box1</Box>
  <Box>...box2</Box>
<Box>
```
