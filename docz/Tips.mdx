# Tips

## 새 컴포넌트를 만들고 싶어요!

![alt](https://res.cloudinary.com/ds9rfw9oy/image/upload/v1534226030/Documents/npmrunnew.gif)

소스 폴더내의 어느 위치에서든 다음 커맨드를 입력합니다.
`npm run new`

생성하려는 컴포넌트의 타입과 이름을 입력한 뒤, malta/src/.. 하위 폴더내에 생성된 소스 두개 .mdx, .tsx를 수졍하면 됩니다.
코딩이 완료됐으면 `npm run build`로 빌드 한뒤 `npm run docz`로 확인합니다.
